Urban Harvest Hub PWA Icon Guide

This project includes SVG icon files for assignment demonstration:
- /pwa-192.svg
- /pwa-512.svg
- /pwa-maskable.svg

For the strongest real-device install support, replace these with PNG files using the same names and sizes:
- 192x192
- 512x512
- 512x512 maskable

Recommended export settings:
- background colour: #2f6b4f
- accent colour: #c77643
- safe padding for maskable icon: at least 20%

If you export PNG icons later, keep the manifest file paths the same so no code changes are needed.
